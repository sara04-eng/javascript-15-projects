# JavaScript 15 Projects

This repository contains **15 JavaScript mini projects** built using **HTML5, CSS3, and Vanilla JavaScript** as part of **Task 11 – Part 2**. Each project focuses on a different JavaScript concept, including DOM manipulation, event handling, timers, animations, and interactive user interfaces.

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6)

---

## Projects Overview

### 1. Color Flipper
A simple project that changes the page background color and displays the selected color whenever the user clicks a button.

### 2. Counter
An interactive counter that allows users to increase, decrease, and reset the displayed value.

### 3. Reviews
A review slider that displays different user profiles and testimonials with navigation controls.

### 4. Navbar
A responsive navigation bar that can be expanded or collapsed on smaller screen sizes.

### 5. Sidebar
A sidebar menu that can be opened and closed using JavaScript for better navigation.

### 6. Modal
A modal popup window that opens and closes through user interaction.

### 7. Questions
An FAQ section where users can expand or collapse questions to reveal their answers.

### 8. Menu
A dynamic restaurant menu that filters items based on selected food categories.

### 9. Video
A custom video player featuring play/pause controls, progress tracking, playback speed adjustment, and mute functionality.

### 10. Scroll
A smooth scrolling webpage with a fixed navigation bar and a scroll-to-top button.

### 11. Tabs
A tab-based interface that displays different content sections without reloading the page.

### 12. Countdown
A live countdown timer displaying the remaining time until a specific future date.

### 13. Lorem Ipsum
A text generator that creates random Lorem Ipsum paragraphs based on the user's input.

### 14. Grocery
A grocery list application that allows users to add, edit, delete, and clear shopping list items.

### 15. Slider
An interactive image slider that allows users to navigate through multiple slides using previous and next controls.
